# Orion
##3D Model Visualizer for Desktop
Orion aims to be a desktop application which allows people to load 3D models and view them without the unnecessary overhead of 3D creational applications (Blender, CAD, Maya etc.).
The project is based on the wildly popular website Sketchfab, a 3D modeling platform website to publish, share, discover, buy and sell 3D, VR and AR content. It provides a viewer based on the WebGL and WebXR technologies that allows users to display 3D models on the web.

##Development
Orion is a personal project being developed solely by myself. It is built using C++ and OpenGL. There are currently no prospects for contributions from friends or the open-source community.

##Features to Come
The aim is to have Orion support these features by the end of 2022:
* Ability to load file formats like: FBX, GLTF, blender
* View models as: Wireframe, with PBR Materials, Diffuse Color, Matcaps
* Support perspective, orthographic and first-person camera views
